Curve Sketching

  1. ANALYZE $f(x)$
    1. Identify the Domain of $f(x)$: Commonly, $f(x)$is undefined for at any x value where:
      1. A denominator equals zero
      2. There is an even root of a negative number
      3. There is a log of a negative number or log of zero
    2. Identify x-intercepts and y-intercept of $f(x)$
      1. x-intercepts: Set $y = 0$ and solve for x
      2. y-intercepts: Set $x = 0$ and solve for y
    3. Identify Vertical Asymptotes and Holes of $f(x)$
      1. $f\left( x \right) = \frac{n}{0}\,\,n \ne 0$ vertical asymptote at x
      2. $f\left( x \right) = \frac{0}{0}$ hole at x
    4. Identify Horizontal Asymptote of $f(x)$
      1. To find the horizontal asymptote, calculate $\mathop {\lim }\limits_{x \to \infty } f\left( x \right)$
  2. ANALYZE $f'(x)$
    1. Graph the critical numbers on a number line and determine the sign for each interval. Separate the number line into intervals with breaks at:
      1. Values of $x$ where$f'(x)=0$
      2. Values of x where $f'(x)$ is undefined
      3. Values of x where $f(x)$ is undefined
    2. Determine the intervals on which $f(x)$ is increasing /decreasing
      1. Test one point contained in the interval (do not use end points of the interval).
      2. $f'(x)<0$ means $f(x)$ is DECREASING on the interval
      3. $f'(x)>0$ means is $f(x)$ is INCREASING on the interval
    3. Identify local maxima and minima of f using first derivative test.
      1. Local Maximum: INCREASING → $f'(x)=0$ → DECREASING
      2. Local Minimum: DECREASING →$f'(x)=0$ → INCREASING
  3. ANALYZE f’’(x)
    1. Graph the critical numbers on a number line and determine the sign for each interval. Separate the number line into intervals with breaks at:
      1. Values of $x$ where $f"(x)=0$
      2. Values of$x$ where $f"(x)$ is undefined
      3. Values of $x$ where $f"(x)$ is undefined
    2. Determine the intervals on which f is concave up/concave down
      1. Test one point contained in the interval (do not use end points of the interval).
      2. $f"(x)<0$ means $f(x)$ is CONCAVE DOWN on the interval
      3. $f"(x)>0$ means $f(x)$ is CONCAVE UP on the interval
    3. Identify inflection points of $f(x)$. A Point of Inflections occurs if one of the following sequences occurs in the intervals:
      1. CONCAVE UP → $f"(x)=0$ → CONCAVE DOWN
      2. CONCAVE DOWN → $f"(x)=0$ → CONCAVE UP

Algorithm for determining Extreme Values

Suppose that $f(x)$ is a continuous function over a closed interval [a, b].

To find the absolute maximum and minimum values of the function $f(x)$on [a, b]:

  1. Find $f'(x)$
  2. Determine the critical points of in [a, b]

    (that is, find all points c for which or does not exist)

  3. List the critical points of $f(x)$ and the endpoints of the interval [a & b]
  4. Find the function values of critical points and the endpoints
  5. The largest of these values is the absolute maximum of on the interval [a, b].
  6. The smallest of these is the absolute minimum of on the interval [a, b].
